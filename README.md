# A Little DB Manager
A little DB manager to manage more easily databases in many projects.

**Works only for MySQL for the moment.**

## FEATURES :

* SHOW TABLES
* SHOW ENTRIES WITH NAME FIELDS
* ADD ENTRIES
* MODIFY ENTRIES
* DELETE ENTRIES

## IN COMMING DAYS :

* ADD / ALTER / DROP TABLE
* ADD / ALTER / DROP FIELD
* BETTER SECURITY
